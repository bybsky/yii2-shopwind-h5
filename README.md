## 项目介绍
ShopWind是一款基于Yii2.0框架深度重构的B2B2C多用户商城系统，微商城/H5版本基于Apache License 2.0开源协议，前后端代码开源无加密，支持二次开发，支持商用。让您快速、无成本搭建起自己的微商城和H5多商户商城。

**ShopWind无偿供大家使用，您的star是我们前进的动力** 

## 运行环境

- Nginx/Apache/IIS
- PHP7.0+
- MySQL5.1+

  建议环境Linux+Nginx+PHP7.0+MySQL5.5.29 以上
